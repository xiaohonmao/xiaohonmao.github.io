    对象的浅复制

    第一种

    只能解决对象第一层属性的所有复制

    给obj1对象添加设置key为prop（键的变量）的键值对

    obj[prop]获取了obj中prop键对应的值

    var obj={a:1,b:2,c:3,d:{a:1}};
var obj1={};
for(var prop in obj){
    obj1[prop]=obj[prop];
}

    第二种

    创建一个空对象，并且将obj的所有属性复制给这个空对象，然后返回给变量obj1

    var obj={a:1,b:2};
var obj1=Object.assign({},obj);

    垃圾回收机制

    所有的局部变量使用完成后都会被自动销毁

    全局变量，如果是number、string、boolean、undefined都会被自动回收

    对于对象而言，所有引用该堆中引用地址的键或者变量都设置为null时，改对象才会被垃圾回收

    函数

    使用function方式定义命名函数，在执行该script标签时，先将所有该标签中描述的函数全部存储在堆中，并且建立栈中的引用关系

    函数也是对象，该引用名就是函数名，而且被存储在window对象下

    函数定义只能定义在调用该函数的标签内任意位置或者这个标签前任意script标签中，不能写在该标签后面的script标签中

    在JS中，如果函数名不是构造函数，首字母小写，单词以驼峰式命名，构造函数首字母大写

    匿名函数必须在var定义之后才可以执行

    var fn=function(){}; fn();

    自执行函数只能执行一遍

    (function(){})();

    构造函数法定义

    var fn=new Function("a","b","console.log('aa')"); fn();

    函数参数

    参数就是函数执行时需要内容，由外部带入

    如果传参是值 实参，如果传参传的是变量 形参

    变量作用域

    变量作用域——变量在什么地方可以被调用

    在函数外是不能够获取函数内的局部变量

    局部变量的作用域仅在该函数内部可以使用，当函数执行完成后，就会清除

    在函数中或全局中this指向window

    ​ 在函数中只要使用var定义了变量，这个变量在整个函数的任何地方都是局部变量，如果有同名的全局变量，将不会在这里被调用

    var a=3;
fn();
function fn(){
    a++;
    console.log(a);    //NaN
    var a=5;
}

    ​ 在ES5中，如果局部变量和全局变量重名时，可以使用window.变量名获取全局变量

    var a=3;
fn();
function fn(){
    var a=5;
    console.log(a+window.a); //8
    console.log(a+this.a);    //8
    console.log(a);    //5
}

    传参过程中：

    ​ 带入非对象时，函数执行不会影响这个形参的改变

    ​ 带入对象时，如果函数中重新设置该参数为一个新对象时，不再修改原有形参引用地址的内容

    ​ 带入对象时，如果函数中直接修改该参数的属性值时，会引起形参的内容改变

    var obj={n:1};
function fn1(a){
    a={n:3}; //重新赋值新对象就是重新设置引用对象地址
}
fn1(obj);
console.log(obj); //{n:1}
    var obj={n:1};
function fn1(a){
    a.n=3; //修改这个地址下n的值，原地址没有修改
}
fn1(obj);
console.log(obj); //{n:3}
    var obj={n:1};
function fn1(a){
    a=5;
}
fn1(obj);
console.log(obj); //{n:1}

    函数参数值的判断

    判断b是不是6种情况，如果是就设置默认值0

    判断a,b是否不是数值，如果不是就设置为默认值0

    如果没有填入b的参数，就让b有一个默认值

    当有参数希望设置为默认值时，尽量将必填参数写在前面，有默认值或者选填的参数写在后面

    函数参数不定值

    arguments 只能在函数中调用，可以获取到所有的参数内容

    arguments.callee; 这是我们当前的函数

    arguments.callee.caller; 执行当前函数的外部函数（回调当前函数的函数）

    函数回调

    把A函数当做B函数参数放入，当运行需要时，在B函数中执行这个参数就可以达到执行A函数的目的

    return

    跳出函数，不执行后面的语句（只能跳出当前函数）

    当在case中使用了return，就不用写break了

    return value 返回一个值，表示该函数执行后会获得一个值

    如果没有使用return返回任何内容，函数执行完成后会获得一个undefined

    return 只能返回一个值

    可以返回函数

    可以返回另一个函数的返回值
