    • 递归

    在函数内，自己执行自己，将自己复制一个副本，执行这个副本

    不停的执行该函数时，就会造成永久性执行（死循环）

    因此，递归必须要有一个条件判断的出口，而且必须使用return跳出

    或者可以当满足条件时，执行自己的函数

    递归解决的是深度遍历，循环解决的是广度遍历

    数组基础

    数组是按照顺序存储一些无序列的元素

    定义方法

    字面量定义
    var arr=[0,1,2,3];
    构造函数定义

    ​ 通过构造函数创建数组时，如果参数仅有一个，并且这个参数是数值时，参数作为数组长度

    ​ 即定义一个具有该数值长度的空数组

    ​ 但是如果这个数值是小数或者负数就会报错

    var arr=new Array(1,2,3,4,5);

    如果参数不是数值就会添加在元素的第一位

    var arr=new Array("3.5");
    数组是对象扩展出来的类型

    ​ 数组也是对象，因此也是引用关系，更改数组的元素，也会引起其他引用改地址的变量产生变化

    var arr1=new Object([1,2,3,4,5]);
var arr4=arr1;
arr1[0]=100;
console.log(arr4);    //[100,2,3,4,5]

    length

    数组的长度就是数组的元素个数

    length是最后一个元素下标+1

    arr[arr.length]=30;    
//在数组的尾部添加一个新元素
//添加元素后arr.length会自动发生改变
    arr.length=0;    //清空数组

    判断是否为数组类型

    Object.prototype.toString.call(arr);
arr.constructor===Array; //构造函数
Array.isArray(arr);    //判断是否是数组（ES6）

    将类数组（列表）转换为数组

    Array.from(); //ES6
Array.prototype.slice.call(arguments);
Array.prototype.concat.apply([],arguments);

    数组方法（上）

    concat

    数组A.concat(数组B)；连接数组，并将连接后的新数组返回

    var arr4=arr1.concat(); 
//如果不填写参数或者填写参数是空数组，都会将数组复制形成新数组返回
    var arr5=arr1.concat(10,20);
//如果直接在参数中填入内容，而非数组，就可以将这些内容添加在新数组的尾部
    arr1=arr1.concat(10,20);
//产生新数组将原来的arr1覆盖了，并且在尾部添加了10，20
    var arr6=arr1.concat(arr2,[10,20]);
//如果写入的参数是多个数组，就可以将多个数组和当前数组合并形成一个新的数组
var arr6=arr1.concat(arr2,10,20);
//如果添加数组和元素混合和上面相同
    var arr7=arr1.concat({a:1,b:2});
//连接对象是把对象当成元素放入
    var arr=Array.prototype.concat.apply(arr1.arguments);
//数组和类数组的连接

    数组也是对象，因此也是引用关系，更改数组的元素，也会引起其它引用改地址的变量展示发生变化

    var arr4=arr1;
arr1[0]=100;
console.log(arr4);

    join

    var arr=[1,2,3,4,5];
//将数组中的元素通过连接字符连接形成新的字符串
var str=arr.join("-");
var str=arr.join("|");
    var str=arr.join();
//默认空时使用.连接
    var str=arr.join("");
//将每个字符紧密连接在一起

    push

    在数组的尾部添加一个或者多个元素，并且返回该数组的新长度

    var len=arr.push(50,10);
//如果添加的是数组，将会把数组作为元素添加在尾部

    unshift

    在数组的头部添加一个或者多个元素，并且返回该数组的新长度

    var len=arr.unshift(0,-1);
//效率低下，因为每次插入数据，数组中的所有值都向后移一位

    pop

    var item=arr.pop();
//删除数组的最尾部一个元素，并且返回被删除的元素

    shift

    var item=arr.shift();
//删除数组的最前面一个元素，并且返回被删除的元素

    数组方法（下）

    splice

    给数组添加/删除/替换一个或多个元素，并且返回被删除的元素组成的新数组

    arr.splice(开始位置，删除多少个元素，添加什么元素)；

    var arr1=arr.splice(); //重建一个空数组
    var arr1=arr.splice(0); 
//将数组的所有元素转移到另一个数组中，本数组变为空
    var arr1=arr.splice(-1); 
//负数是指从后向前，-1表示仅删除最后一个并返回到新数组中
    var arr1=arr.splice(-2); 
//删除最后两个并返回这个被删除的数组（将最后两位移到新数组中）
    var arr1=arr.splice(0,1); 
//表示删除数组的第0位元素，并且返回到新数组中（将数组的第0位移到新数组中）
    var arr1=arr.splice(0,2);
//表示将前两位移动到新数组中
    var arr1=arr.splice(2,2,6,7); 
//表示从下标为2开始删除后面的两个元素，返回到新数组中，并且在这个位置添加6,7
//意味着将数组的第二位及以后的两个元素替换为6和7

    slice

    slice 截取，将数组的从指定的开始位置到结束位置截取复制返回新数组，原数组不变

    arr.slice(开始位置，结束位置)

    var arr1=arr.slice(1,3);
//从数组的第一项开始截取到第三项之前
    var arr1=arr.slice();
//复制arr数组到arr1中，没有引用关系，改变一个的值，另一个不会改变
var arr1=arr.slice(0);
//同上
    var arr1=arr.slice(-1);
//复制数组的最后一个元素到新数组
    var arr1=arr.slice(2，-1);
//从下标为2到倒数最后一个之前复制
    var arr1=arr.slice(-3,-1);
var arr1=arr.slice(1,3);
//保证在开始位置和结束位置是统一的正数或者负数，然后开始位置必须小于结束位置

    indexOf

    在数组中查找元素的位置（下标），如果没有找到返回-1

    indexOf(查找的内容，从第几个开始查找)

    lastIndexOf(); 从后向前查找

    forEach

    遍历arr数组，执行里面的函数，传入参数第一个参数是数组的每个元素，第二个参数是每个元素的下标，第三个参数是数组自身

    map

    使用map遍历，可以使用return返回新数组的元素，最后形成一个新数组

    map遍历返回时，原数组的长度和新数组的长度完全相同

    不能作为筛选使用，因为map遍历返回的数组和原数组长度相同

    reverse

    倒序 将数组倒序，返回原数组，改变原数组

    sort

    采用冒泡算法的原理

    先将数组中的元素转换为字符串，然后将该字符串的第一个字符取出，在转换为Unicode编码，然后比较

    如果两个被比较的第一个字符Unicode编码相同时，会将下一个字符转换成Unicode编码在进行比较，直到完毕。

    var arr=[2,5,1,3,4,7,9,8,6,10,13];
arr.sort(function(a,b){
    return a-b;
});
//a-b    从小到大排序
//b-a    从大到小排序

    reduce

    arr.reduce(fn,initValue); 第一个参数是回调函数，第二个参数是初始值

    回调函数中有四个参数，第一个是上次归并的结果，第二个是遍历的元素，第三个是下标，第四个是数组

    没有initValue（初始值）时

    ​ 第一次进入遍历时，value就是数组的第0项，item则是第1项，从第1项开始遍历数组到尾部

    ​ 如果没有使用return返回值，在下一次遍历时，归并值就是undefined

    如果有initValue时

    ​ 第一次进入的value就是initValue的值，遍历则从第0项开始

    some

    数组中判断是否有满足条件，如果有则返回true，并且不再继续遍历判断

    every

    数组中判断是否没一个都满足条件，如果有一个是不满足条件的，则不继续遍历，直接返回为false

    filter

    return返回满足条件的元素

    flatMap

    扁平化数组 是指将一个多维数组变为一维数组

    桥接模式

    forEach、map、sort、reduce、some、every、filter、flatMap

    数组遍历和二维数组

    数组是一个紧密型的数据结构

    数组元素有属性，for in会遍历到属性，for语句不会，forEach语句也不会遍历属性

    数组中有空元素，for in不会遍历空元素，for语句会遍历空元素，forEach不会遍历空元素

    当使用for in遍历时，索引值都会当成对象的属性来遍历，因此索引值会变成字符类型

    内建对象

    Math是一个静态方法集

    Math.abs(); //求绝对值

    求整数的输入

    Math.ceil(); //向上舍入
Math.floor(); //向下舍入
Math.round(); //四舍五入
    Math.max(3,6,9); //最大值
Math.max.apply(null,arr); //求数组的最大值
Math.min(3,6,9); //最小值
Math.min.apply(null,arr); //求数组的最小值
    Math.sqrt(4); //求平方根
    Math.pow(2,128); //求2的128次幂
Math.pow(4,1/2); //效率比sqrt要低
Math.pow(4,1/3); //求立方根
    Math.random(); //随机值0-1之间，不包含0和1

    String基础

    所有的字符串方法都不会改变原字符串

    var str="abcdef"; //string 存储在栈中
var str=new String("abcdef"); //Object 存在堆中
str[0]; //str字符串中第0位的字符

//不能设置和改变该字符串中对应位置的字符内容
str[0]="z"; //不可修改，只能获取
    str.length  //获取字符串的长度，包含了多少个字符，中文单字长度也是1
str.charAt(0); //相当于 str[0]
str.charCodeAt(0); //返回第几个下标对应字符的Unicode编码
string.formCharCode(97); //将Unicode编码转换成字符
indexOf(); //同数组中一样
lastIndexOf(); //同数组中一样
str.search("b"); //主要用在正则表达式中
str.match("a"); //主要用在正则表达式中

    替换、截取、切割

    原字符串不改变

    替换

    str.replace(要查找的字符或字符串|要查找的正则表达式，要替换的字符串|函数);

    截取

    str.substring(start,end); //截取

    //substring负数是0之前的，只要是负数就可以理解为0（负数无效可以理解为0）
str1=str.substring(3,1); //从后向前截取

    str.substr(start,length); //截取

    从第几位开始向后截取多少长度的字符串，第二个参数必须大于0

    str.slice(start,end); //与数组的slice完全相同

    切割

    ste.split(“”); //分割

    其他

    str.toUpperCase(); //转换为大写

    str.toLowerCase(); //转换为小写

    JSON.stringify(obj); //将对象转换为JSON字符串格式

    JSON.parse(str); //将JSON字符串转换为对象

    Number类型

    var a=5; //number 存在栈中
var a1=new Number(5); //Object Number 存在堆中
Number.MAX_VALUE; //1.7976931348623157e+308
Number.MIN_VALUE; //5e-324
Number.POSITIVE_INFINITY; //正无穷 Infinity
Number.NEGATIVE_INFINITY; //负无穷 -Infinity

console.log(a.toExponential(10)); //3.5078600000e-1

    Date类型

    var date=new Date();
date.getTime(); //1970.1.1 到现在的毫秒数，永远不重复，可以用作时间戳
//F5刷新时从缓存中获取数据

    实例

    var img=new Image();
img.src="./img/a.jpg?time=" + (new Date()).getTime();
//保证图片每次都取到最新的，不去缓存中获取

    获取时间

    var date=new Date();
date.getFullYear(); //年
date.getDate(); //日期
date.getMonth(); //月份 0-11
date.getDay(); //星期 0-6 0是周日
date.getHours(); //小时
date.getMinutes(); //分钟
date.getMilliseconds(); //毫秒
date.getseconds(); //秒
date.toUTCString(); //转换为格林尼治时间字符串
date.getUTCHours(); //获取格林尼治时间小时

    设置时间

    设置时间超过最大值就会进位

    date.setHours(25);
date.setMonth(12);
date.setMinutes(date.getMinutes()+30);
