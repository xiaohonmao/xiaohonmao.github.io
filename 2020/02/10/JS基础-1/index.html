    变量和常量

    变量

    这里的变量被存储在栈中，栈中的数据存储内容较少，读取速度较快

    变量名是区分大小写的

    变量名一般都要有一定含义，尽量减少无意义的变量名

    变量名首字母小写，每个单词的首字母大写

    stepValue 采用驼峰式命名

    不能使用数值起头，如果使用“_”起头，意味着这个变量是一个临时变量或者参数

    不能使用关键词或者保留字

    可以在设置以后重新改变这个变量的值

    当定义一个变量时，意味着将这个变量作为window的全局属性定义了

    定义变量时需要避开window下已有的属性名

    常量

    设置以后不能修改常量的值

    常量名称全部大写，以下划线区分单词

    弹框

    alert(); 警告框

    confirm(); 确认消息框

    prompt(); 提示消息框

    一元运算符

    i++; 后置自增：是先取值，在+1（当获取值时，先将“i”的值返回打印出来，在做”i+1”）

    ++i; 前置自增：是先+1，再取值（这里是先把”i+1”，然后将运算的结果返回并打印）

    单独运算时，两个结果是相同的

    一旦使用一元运算符，所有的内容将会隐式转换为数值运算

    强制转换

    字符串

    a=string(a); //强制转换为字符

    toString(进制) 按照进制将内容转换为字符串

    a=a.toFixed(2); //保留小数点后几位，会四舍五入

    数值

    var a=”a3”; a=Number(a); //字符串中有字符时，通过Number强制转换都是“NaN”

    var a=true; //布尔值强转数值后变成0(false)和1(true)

    var a; //undefined类型强转后是“NaN”

    var a=null; //null类型强转后变为0

    var a={a:1,b:2}; //所有对象强转数值变为“NaN”

    a=Number(a); //强制转换为数值

    a=parseInt(a); //强转取整，如果字符串起始是数值，在使用这个转换时，会将这个字符串起始到第一字符位置之前的内容转换为数值，不会四舍五入。

    布尔值如果使用parseInt强转后变为“NaN”

    其他非字符和数值的任何类型通过parseInt强转后变为“NaN”

    a=parseFloat(a); //如果有小数点，转换为浮点数

    NaN 数值类型，非数值，任何一个非数值都和其他非数值不相等

    布尔值

    只有“ ”、0、false、undefined、null、NaN强转布尔值时是false，其它都是true

    隐式转换

    任何数值类型与字符相加时，会自动将数值类型转换为字符类型，然后首尾相连

    所有隐式转换均使用Number()、String()、Boolean()

    加法运算中如果有字符参与，就会隐式转换为字符，再相连；否则如果没有字符参与，将会隐式转换为数值，进行数值相加

    如果减法、乘法、除法、取模运算时，都会隐式转换为数值再运算

    使用一元运算符，所有内容将会隐式转换为数值运算

    条件语句

    if()

    不论给的是什么值，都会强转为布尔值

    如果布尔值是true，进入条件语句

    if(a){}; //a不等于6种情况（null、false、undefined、“ ”、0、NaN）

    if(a==true){}; //“1”、1、true

    if(a===true){}; //true

    if(!a){

    ​ //a是不存在或者没有初始值，或者已经被设为null

    }

    //a是6种情况可以进入

    if(a==false){}; //“ “/、false、0

    if(a===false){}; //false

    if(a+b){};

    if(obj && obj.a){};

    if(obj || obj !=0){};

    条件只能嵌套，不能交叉

    条件语句，如果内容只有一句话，可以省略花括号

    三目：var b=a > 3 ? 10 : 20;

    switch

    switch主要判断是否相等，===比较类型和值

    循环语句

    while

    while(变量的条件){

    ​ //变量变化（必须向条件外变化）

    };

    自动进入判断条件，直到条件为false时，不再进入

    循环语句，如果内容只有一句话，可以省略花括号

    如果遇到continue,跳出当前的循环，继续下一次循环，注意需要将变量变化的表达式写在continue前面，防止不能改变变量造成死循环

    do while

    这里会先执行循环体，在判断条件

    一般用于先需要处理数据，处理完成后再循环执行

    for

    for(var i=0; i<10; i++){};

    第一部分 var i=0; 定义变量，在整个循环中仅执行一次，并且在最开始执行

    第二部分 i<10; 条件判断，在每次循环前做条件判断

    第三部分 i++; 循环语句块全部执行完成以后，需要循环时执行这个语句，完成后接着判断条件

    for(var i=0,sum=0; i<10; i++){};

    只允许使用var定义的变量，let定义的变量sum不能写在里面

    强制记忆

    var b=10; console.log(Object.prototype.toString.call(b)); //[object Number]

    var b=10; console.log(Object.prototype.toString.call(b)); //[object Number]

    var b=10; console.log(Object.prototype.toString.call(b)); //[object Number]

    var e=null; console.log(Object.prototype.toString.call(e)); //[object Null]

    使用Object.prototype上的原生toString()方法判断数据类型

    相关博文：https://www.cnblogs.com/cn-chy-com/p/11524980.html

    对象

    var obj={a:1,b:2};

    对象是一个集合，是由若干个键值对组成

    在一个对象的同层中不允许出现相同的键

    如果键不是字符类型，就会隐式转换为字符类型

    JS是一种点语法语言，点语法语言只能设置指定字符串为关键词的键值对

    []中关键词可以是任何内容，包括字符、变量、表达式

    for(var prop in obj){};

    ​ 遍历查看obj对象中有哪些关键词，并且每次将关键词赋值给变量prop，prop为key的变量

    ​ 遍历中每一个可枚举属性都可以查看到，根据该key创建的先后顺序遍历

    var obj={a:1,b:2}; delete obj.a; //可以删除可删除属性，不能删除冻结对象的属性

    对象存储在堆中

    var obj={a:1,b:2}; var obj1=obj; //仅是将obj的引用地址赋值给obj1
